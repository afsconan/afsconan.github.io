运行程序时，输出目录需不存在 运行 Hadoop 程序时，结果的输出目录（如output）不能存在，否则会提示错误，因此运行前需要先删除输出目录。建议在程序中加上如下代码进行删除，避免繁琐的命令行操作： Configuration conf = new Configuration(); Job job = new Job(conf); ... /* 删除输出目录 */ Path outputPath = new Path(args[1]); outputPath.getFileSystem(conf).delete(outputPath, true); ... ——————————— 首先去Eclipse官网下载 Eclipse IDE for Java Developers 的Mac OS X 64 Bit版本。 然后新建一个项目,然后 Project Properties -> Java Build Path -> Libraries -> Add External Jars 。这里需要添加两个jar包,都在hadoop主目录里。分别是: hadoop􏰇􏰈􏰉/share/hadoop/common/hadoop-common-2.6.0.jar 以及 hadoop􏰇􏰈􏰉/share/hadoop/mapreduce/hadoop-mapreduce-client-core-2.6.0.jar 然后输出为jar包。File -> Export -> Java -> JAR File。然后点击next。选中WordCount源文 件, 设置输出路径和文件名WordCount.jar, 选择Finish则打包成功。

运行程序时，输出目录需不存在
运行 Hadoop 程序时，结果的输出目录（如output）不能存在，否则会提示错误，因此运行前需要先删除输出目录。建议在程序中加上如下代码进行删除，避免繁琐的命令行操作：

Configuration conf = new Configuration();
Job job = new Job(conf);
...
/* 删除输出目录 */
Path outputPath = new Path(args[1]);
outputPath.getFileSystem(conf).delete(outputPath, true);
...
———————————

首先去Eclipse官网下载 Eclipse IDE for Java Developers 的Mac OS X 64 Bit版本。
然后新建一个项目,然后 Project Properties -> Java Build Path -> Libraries -> Add External Jars 。这里需要添加两个jar包,都在hadoop主目录里。分别是:
hadoop􏰇􏰈􏰉/share/hadoop/common/hadoop-common-2.6.0.jar 以及
hadoop􏰇􏰈􏰉/share/hadoop/mapreduce/hadoop-mapreduce-client-core-2.6.0.jar

然后输出为jar包。File -> Export -> Java -> JAR File。然后点击next。选中WordCount源文 件, 设置输出路径和文件名WordCount.jar, 选择Finish则打包成功。
